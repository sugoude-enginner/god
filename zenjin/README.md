- メンター(ZENJIN)⇔リバースメンター(REVERSE ZENJIN)
  - メンターとかリバースメンターになることで孤独から抜け出せる（ZENJINの夜明け）
- 期間アンロック
  - メンターとリバースメンターが関係（ZENJING)を開始して、１週間（お試しとして）経ったら、期間を延長するようなイメージ
- マッチング（ZENCHING）
- 記録機能
